# はんだづけ

向きを間違えやすい箇所を説明します。

U1のCH340は上下2つある白線のうち伸びている箇所(左上)がピン1です。そのため写真のような向きではんだづけしてください。

<img src="./add_info/CH340.JPG" alt="CH340のはんだづけ" width="300"/>  
  
D1のチップLEDは写真のような向きです。チップLEDの裏面のほうが分かりやすいため裏面にしたLEDも横に並べています。
